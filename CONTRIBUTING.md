# Contributing

## Ground rules

- Source-only changes.
- Keep this repo clean-room; do not paste source from projects without a license.
- Prefer small pull requests with tests.
- Data files and translations should be optional and defensively loaded.

## Development

```powershell
dotnet restore
dotnet build
dotnet test
```

## Commit style

Use short, practical messages such as `Add price cache store` or `Add calibration preview models`.
