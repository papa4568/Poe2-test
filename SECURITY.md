# Security Policy

This app captures portions of the user's screen and makes network calls for price data. Treat security reports seriously.

## Supported versions

Only the latest `main` branch is supported until the first public release.

## Reporting

Open a private security advisory if available, or contact the repository owner directly.

## Rules

- No prebuilt binaries in PRs.
- No hidden network calls.
- No telemetry without an explicit setting.
- No credentials, tokens, or personal files in logs/debug bundles.
