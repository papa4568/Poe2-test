# Clean-room notes

This repository is a new implementation and should not copy source code from projects that do not grant reuse rights.

Allowed inputs:

- Our own code.
- Public API documentation.
- Licensed dependencies.
- General product behavior observed as a user.
- Original tests and models written for this project.

Avoid:

- Copying source files from unlicensed repositories.
- Reusing branding, donation links, release feeds, icons, or exact UI copy from another project.
- Adding generated files from unknown sources without review.

The project uses MIT licensing for its own code from the first commit.
